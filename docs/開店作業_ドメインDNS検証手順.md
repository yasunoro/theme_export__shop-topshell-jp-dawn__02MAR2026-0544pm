# ドメイン・DNS 完了検証手順

設定が「正しくできているか」を確認するためのチェック項目と実行方法。

---

## 検証項目一覧

| # | 検証内容 | 合格基準 | 実行方法 |
|---|----------|----------|----------|
| 1 | ストアが HTTPS で応答する | ステータス 200 かつ HTTPS | スクリプト or curl |
| 2 | SSL 証明書が有効である | 証明書エラーなし | ブラウザ or `curl -v` |
| 3 | プライマリ URL でアクセスできる | 想定ドメインでページが表示される | ブラウザ |
| 4 | リダイレクトが意図どおり（任意） | www↔非www が設定どおり | スクリプト or curl -I |

---

## 実行方法

### A. 自動チェック（推奨）

リポジトリ直下で:

```bash
# 使用するストア URL を指定（末尾のスラッシュなし）
export STORE_URL="https://shop-topshell-jp.myshopify.com"
# または独自ドメインの場合
# export STORE_URL="https://your-domain.com"

./scripts/check-domain-dns.sh
```

未指定の場合は `https://shop-topshell-jp.myshopify.com` で実行されます。

### B. 手動チェック

1. **Shopify 管理画面**  
   設定 > ドメイン を開き、「接続済み」「SSL が有効」と表示されているか確認する。

2. **ブラウザ**  
   - ストアの URL を開く。  
   - アドレスバーに鍵マークが出ており、証明書エラーが出ないこと。  
   - トップページが表示されること。

3. **リダイレクト（独自ドメイン利用時）**  
   - `http://` で開く → `https://` にリダイレクトされること。  
   - www / 非 www のどちらを正とするか設定済みなら、もう一方から正の URL にリダイレクトされること。
